Occourance of each character of a string

Solution-1:

      package com.array.Programming;
      public class CharacterOccurrence {

    public static void main(String[] args) {
        String str = "occurrence example"; // Example string
        int[] freq = new int[256]; // Array to store frequency of each character

        // Calculate frequency of each character
        for (int i = 0; i < str.length(); i++) {
            char ch = str.charAt(i);
            if (ch != ' ') { // Ignore spaces
                freq[ch]++;  // Increment frequency of character
            }
        }

        // Print characters and their frequencies
        System.out.println("Character occurrences:");
        for (int i = 0; i < freq.length; i++) {
            if (freq[i] > 0) {
                System.out.println((char) i + ": " + freq[i]);
            }
        }
    }
}

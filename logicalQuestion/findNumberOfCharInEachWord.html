Write a program to find the number of characters in each word of a String?

Input: my name is sonu

Output:

Solution-1:

      package logicalQuestion.string;

      public class FindNumberOfCharEachWord {
          public static void main(String[] args) {
              String s = "my name is sonu";
              StringBuffer str = "";
              int count = 0;
              for (int i = 0; i < s.length(); i++) {
                  if (s.charAt(i) == ' ') {
                      System.out.println(str + ".." + str.length());
                      str = "";
                      count = 0;
                  } else {
                      count++;
                      str = str + s.charAt(i);
                  }
              }
          }
      }

Solution-2:

      package logicalQuestion.string;

      public class FindNumberOfCharEachWord {
          public static void main(String[] args) {
              String s = "my name is sonu";
              String[] words = s.split("\\s");
              for (String word : words) {
                  int length = word.length();
                  System.out.println(word + " character length is: " + length);
              }
          }
      }

Lambda Expression

  • LISP is the first programming language that supported lambda expressions.
    • LISP
    • C
    • C++
    • C#
    • Scala
    • Ruby
    • Python, etc.
  • Java started supporting Lambda expressions from version 1.8 (Java is late in adopting lambdas).
  • Java is an object-oriented programming language. By introducing Lambdas, it now supports functional programming as well.

Object-Oriented Programming vs Functional Programming?

  • In Object-Oriented Programming Languages, Classes & Objects are the main entities.
  • If we want to create a function in OOP languages, that function should be created inside the class.
  • In Functional Programming, a function can exist outside the scope of an object.
  • We can assign a function to a reference variable.
  • One function can be passed as a parameter to another function.

What is Lambda?

  • A Lambda Expression is just an anonymous function.
    1. No Name
    2. No Modifier
    3. No Return Type

Why Lambda Expression?

  • Code Optimization
  • Readability
  • Less Coding
  • To write functional programming in Java
  • To write more readable, maintainable, and concise code

How to write Lambda Expression?

  • Writing Lambdas is very easy if you know how to write methods in Java.
  • Let us see one example to understand how to write lambda.

Example:-1

      public void greet() {
        System.out.println("Hi");
      }

After Java 8

      () -> System.out.println("Hi");

Example:-2

      public void add(int a, int b) {
        System.out.println(a + b);
      }

After Java 8

      (a, b) -> System.out.println(a + b);

Example:-3

      public User getUserById(Integer userId) {
        // logic to get from DB
        return user;
      }

After Java 8

      (userId) -> {
        // logic to get from DB
        return user;
      }

Example:-4

      public User getUserById(int userId) {
        UserDao dao = new UserDao();
        User user = dao.findById(userId);
        return user;
      }

After Java 8

      (userId) -> new UserDao().findById(userId);

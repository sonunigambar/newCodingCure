What is Supplier

  • It is a predefined functional interface.
  • It contains only one abstract method i.e get().
  • Supplier is an interface that does not take any arguments but it produces a value when the get() function is called.
  • Suppliers are useful when our lambda is not taking any input but it is providing some output.

Example 1:

public class DateApp {
    public static void main(String[] args) {
        Supplier s = () -> new Date();
        System.out.println(s.get());
    }
}

Example 2:

public class OtpGenerator {
    public static void main(String[] args) {
        Supplier s = () -> {
            String otp = "";
            for (int i = 1; i <= 6; i++) {
                otp = otp + (int) (Math.random() * 10);
            }
            return otp;
        };
        System.out.println(s.get());
        System.out.println(s.get());
        System.out.println(s.get());
    }
}
